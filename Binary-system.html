Binary System

Binary number is a number expressed in the base-2 numeral system or binary numeral system, a method of mathematical expression which uses only two symbols: typically "0" (zero) and "1" (one).

  • The base-2 numeral system is a positional notation with a radix of 2.
  • Each digit is referred to as a bit, or binary digit.
  • The modern binary number system was studied in Europe in the 16th and 17th centuries.
